Explication des balises utilisées dans le code

Dans ce code, plusieurs balises ont été choisies pour structurer un CV :

  • <table> et <td> : Organiser les informations dans un tableau.
  • <header> : Regrouper les informations importantes comme le nom, les coordonnées.
  • <ul> et <li> : Créer des listes à puces pour les compétences, expériences, etc.
  • <a> : Insérer des liens comme l'email et l'URL vers la chaîne YouTube.
  • <img> : Afficher l'image de profil.
  • <h1>, <h2>, <h3> : Organiser le contenu en titres et sous-titres.
  • <meta> : Définir les informations de métadonnées.
